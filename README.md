# expansion.h 

This is a experimental project created for learning and practice. 
It is a single header utility for The C Programming Language which includes:
- String implementation
- Dynamic Arrays
- Arena Allocators

It's likely not suitable for your project and I don't recommend using it.

### This code is a work in progress and may change.
It is not fully optimized or fully tested as it is just used for my personal projects.
